## Warehouses Feature

- **Purpose:** Track warehousing workflows for China and Libya locations; surface inventories and status workflows.
- **Structure:**
  - `slices/` caches warehouses metadata/state.
  - `hooks/` provide filtering/search helpers.
  - `components/` will show dashboards, location cards, and status badges.
  - `api/` ties to `src/app/api/warehouses`.
